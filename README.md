## ✨ 作品展示：個人作品集網站

`rain-yu-portfolio.html` 是課程的實作成果，包含以下區塊：

- **Header** — 導覽列，含 Logo 與頁面錨點連結
- **About** — 個人介紹、大頭照，以及 Instagram / GitHub / HackMD 社群連結
- **Interests** — 三欄式興趣卡片（美食 🍣 / 遊戲 🎮 / 音樂 🎵），具 hover 動畫效果
- **Footer** — 版權聲明

## 🚀 如何在本地端執行

不需要任何安裝！直接用瀏覽器開啟即可：

```bash
# 1. Clone 這個專案
git clone https://github.com/RainYu0510/Web-Course.git

# 2. 進入資料夾
cd Web-Course

# 3. 直接用瀏覽器開啟
open rain-yu-portfolio.html
# 或直接雙擊檔案
```
